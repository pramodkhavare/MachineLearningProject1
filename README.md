# Docker file command
```
docker status

To list docker image
```
docker images
```

Run Docker Image
```
docker run -p 5000:5000 -e PORT=5000 <Image_id>
```

To check running docker container
```
docker ps
```

To stop docker container
```
docker stop <container_id>
